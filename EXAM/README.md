# EXAM Project

This folder contains a student performance and wellbeing analysis project, including data generation, cleaning, and a final notebook for modeling and evaluation.

## Contents

- `student_performance/` — contains all files related to the student wellbeing project.
- `student_performance.zip` — archived version of the student performance folder.

## Student Performance Project Summary

The project focuses on generating and analyzing student wellbeing data, performing data cleaning, and building models to understand student performance outcomes.

### Directory: `student_performance`

Key components:

- `data_generation_code.py` — Python script used to generate synthetic student wellbeing and performance data.
- `student_wellbeing_final.csv` — original dataset containing student wellbeing and performance features.
- `student_wellbeing_final_cleaned.csv` — cleaned version of the dataset ready for modeling.
- `student_wellbeing_final.ipynb` — Jupyter notebook that explores data preprocessing, analysis, and model training.

Key Topics: Data generation, exploratory data analysis, data cleaning, feature preparation, and modeling for student wellbeing or performance prediction.

## How to Use

1. Unzip `student_performance.zip` if needed.
2. Open `student_performance/student_wellbeing_final.ipynb` in Jupyter Notebook or JupyterLab.
3. Run the notebook cells sequentially to follow the data analysis and modeling workflow.
4. Review the generated datasets, model results, and conclusions.

## Requirements

The notebook and script uses standard Python libraries such as:

- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn

Install dependencies with:

```bash
pip install pandas numpy matplotlib seaborn scikit-learn
```
