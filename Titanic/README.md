# Titanic Survival Analysis

This folder contains a Titanic dataset notebook that demonstrates data exploration, preprocessing, model building, and evaluation for predicting passenger survival.

## Contents

- `titanic.ipynb` — Jupyter notebook covering the Titanic machine learning workflow.

## Notebook Summary

The notebook includes:

- Loading the Titanic dataset and inspecting its structure
- Exploratory data analysis (EDA) with summary statistics and visualizations
- Cleaning and preprocessing data, including handling missing values and encoding categorical features
- Feature selection and preparation for modeling
- Training classification models to predict passenger survival
- Evaluating model performance with metrics such as accuracy, confusion matrix, and classification report

## Requirements

The notebook uses common Python libraries such as:

- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn

Install dependencies with:

```bash
pip install pandas numpy matplotlib seaborn scikit-learn
```

## How to Use

1. Open `titanic.ipynb` in Jupyter Notebook or JupyterLab.
2. Run cells sequentially from top to bottom.
3. Review the data preprocessing steps, model training output, and evaluation results.
